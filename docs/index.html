[동국아 졸업하자!] 앱이 왜 필요할까요?

학교는 열심히 다닌것 같은데...졸업까지 뭘 더 해야하는지 모르겠는 대학생들
나의 남은 졸업요건을 쉽게 확인할 수 있다면?

계획을 짜고는 싶은데 방대하고 복잡한 졸업요건 카탈로그에 지친 대학생들
나에게 필요한 졸업요건을 쉽게 확인하고 장기적인 계획까지 짤수 있다면?

졸업계획



  • 이수체계도를 매번 찾아보면서
    졸업계획을 짜기 힘드시죠?

    이제는 보다 쉬운 졸업이수체계도
    확인을 통해 더욱 쉽고 빠르게
    졸업 계획을 세울수 있습니다.


    또한

    나만의 졸업계획을 통해 졸업하기까지의
    체계적인 계획을 짤 수 있습니다.

졸업요건



  • 복잡한 졸업요건 카탈로그를
    일일이 확인하기 어렵지 않나요?

    학과와 학번만 선택하면
    사용자에게 필요한 졸업요건 정보를
    보다쉽게 확인할 수 있습니다.



    현재 자신의 남은 졸업학점을
    알고계신가요?

    나만의 졸업계획과 충족요건입력을
    통해 자동으로 남은 요건이 계산되어
    한눈에 파악이 가능합니다.
